# Cart-React

A simple React shopping cart app.

## Features
- Add items to cart
- Remove items
- Update quantities
- Show total price

## Tech
- React
- JavaScript
- CSS

## Setup
1. Clone repo: `git clone https://github.com/sofyan985/cart-react.git`
2. Go to folder: `cd cart-react`
3. Install: `npm install`
4. Run: `npm start`

## Usage
Open browser to view app. Add/remove items in cart.
